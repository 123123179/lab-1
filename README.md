# Лабораторна робота №1. Робота з HTTP протоколом, запитами та публічними API 

У межах лабораторної роботи, виконано ознайомлення з принципами роботи HTTP-протоколу та публічних REST API на прикладі сервісу Національного банку України.

Згідно завдання 1, було здійснено GET-запит до API НБУ за допомогою інструмента Postman з метою отримання курсу долара США (USD) за визначений період. У відповіді сервера було отримано дані у форматі JSON, що підтверджує коректність сформованого HTTP-запиту як це зображено на рисунку 1.

У завданні 2 реалізовано отримання курсу валюти з використанням мови програмування Python та бібліотеки requests.
